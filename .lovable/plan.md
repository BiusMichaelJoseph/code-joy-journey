

# Code Joy Academy — Website Plan

## Overview
A professional, mobile-responsive lead-generation website for Code Joy Academy — a coding and education academy. The site captures inquiries via an external Google Form (no payments, no e-commerce). Bright, friendly design with blues, greens, and white.

---

## Pages

### 1. Homepage
- **Hero Section** with headline (*"Unlock the Joy of Coding — From First Line to First Job"*), subheadline, and CTA buttons ("Enroll Your Child", "Explore Courses", "Book Free Consultation") linking to the Google Form or Services page
- **Key Benefits** strip — e.g., Project-Based Learning, Certified Instructors, All Ages Welcome
- **Course Categories Preview** — cards for Junior Coders, Teen Developers, Adult Track, Holiday Bootcamps, 1-on-1 Tutoring — each linking to the Services page
- **Testimonials Section** — placeholder cards with parent/student quotes
- **How It Works** — simple 3-step visual (Inquire → Get Matched → Start Learning)
- **Final CTA Banner** driving visitors to the enrollment form

### 2. Services Page
- Detailed sections for each program category:
  - Junior Coders (Ages 7–10)
  - Teen Developers (Ages 11–17)
  - Adult & Career Track
  - Holiday Bootcamps
  - 1-on-1 Tutoring
- Each section includes: description, topics covered, delivery format (online/in-person/hybrid), and what's included (certificates, portfolio, community)
- No pricing shown — each section has a CTA button linking to the Google Form
- Additional programs: Homeschooling Support, Exam Preparation, Special Needs Support

### 3. About Page
- Mission statement and founding story
- Teaching philosophy (project-first, low stress, high support)
- Instructor credentials placeholder
- Diversity & inclusion commitment
- Optional: team member photo placeholders

### 4. Enrollment / Contact Page
- Brief intro text explaining the enrollment process
- Prominent embedded Google Form **or** a large CTA button linking to the external Google Form
- Contact details (email, phone/WhatsApp, location if applicable)
- Note: "Our team will reach out within 24 hours"

---

## Shared Components
- **Navigation Bar** — Logo, links to Home, Services, About, Enroll (highlighted CTA button), mobile hamburger menu
- **Footer** — Links to all pages, social media icons (Instagram, Facebook, WhatsApp), contact info, copyright
- **Consistent CTA Buttons** throughout the site pointing to the Google Form

---

## Design Direction
- **Colors:** Blues, greens, white with friendly accents
- **Style:** Modern, clean, approachable — suitable for parents and students of all ages
- **Typography:** Clear, readable fonts
- **Mobile-first** responsive layout
- **Fast-loading** static site with no backend required

---

## Technical Notes
- No backend/database needed — purely a frontend site
- Google Form linked externally via CTA buttons (user provides the Google Form URL)
- No payment integration
- All content is static with placeholder data that the user can update
